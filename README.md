Welcome to the **Sarah Flynn Design** portfolio source code. This project features a clean, highly aesthetic, and premium portfolio layout designed for a creative professional.

## 🚀 Features
- **Aesthetic UI/UX**: Soft colors, glassmorphism, and elegant typography (Playfair Display & Inter).
- **Smooth Animations**: High-performance scrolling and hover effects to make the portfolio feel alive.
- **Responsive Layout**: Custom CSS grid and flexbox logic ensuring it looks stunning on both desktop and mobile.
- **Premium Chatbot Widget**: Integrated an interactive floating chat interface for quick client contact.
- **Custom Cursor**: A beautiful floating custom cursor for that extra touch of luxury.

## 🛠️ Technology Stack
- **HTML5 & Vanilla CSS3**
- **Vanilla JavaScript** (for interactive components and chatbot logic)
- Zero external libraries for maximum performance and lightning-fast loading speeds.

## 💻 How to Run Locally
If you want to run the raw code on your machine:
1. Clone this repository.
2. Open the directory in your terminal.
3. Run `npx serve` to start a local server.
4. Visit `http://localhost:3000`.
